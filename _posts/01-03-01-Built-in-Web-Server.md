## 內建的網頁伺服器 {#builtin_web_server_title}

從 PHP 5.4 開始, 你可以不需要安裝與設定一個完整的網頁伺服器. 你只需要在終端機程式中, 於你的專案目錄下執行以下指令就可以啟動內建的伺服器:

{% highlight console %}
> php -S localhost:8000
{% endhighlight %}
